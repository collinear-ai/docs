Collinear's Agentic AI solution accepts workflow logs in multiple formats to begin the evaluation process.

### Supported Formats
- Text logs (`.txt`, `.log`)
- CSV files (`.csv`)
- JSON data (`.json`)
- Conversation transcripts
- Custom agent workflow exports

### How To Upload

1. Navigate to the Agentic AI section in your workspace
2. Click "Upload Workflow Logs"
3. Select your file(s) or drag-and-drop
4. Specify workflow type (chat, automation, etc.)
5. Confirm upload

### Best Practices
- Ensure logs contain complete workflow sessions
- Include timestamps where available
- Remove sensitive data before uploading
