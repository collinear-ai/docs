Collinear's Agentic AI solution enables you to:

1. Upload agent workflow logs in various formats (text, logs, CSV, JSON, conversations) as input
2. Automatically generate evaluation data for speciffic agent workflows
3. Assess agentic use cases on generated data and custom metrics
4. Review and export results

## Key Features

- **Input Flexibility**: Works with multiple file formats
- **Evaluation Data Generation**: Create realistic data based on your inputs
- **Comprehensive Evaluation**: Assess data across three dimensions:
  - Safety
  - Reliability
  - Performance
