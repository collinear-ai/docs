---
title: "Authentication"
description: "API key setup and authentication for the SimLab API"
---

All API requests require an API key passed via the `API-Key` header.

```
API-Key: <your-api-key>
```

## Obtaining a Key

```bash
simlab auth login
```

This will direct you to [platform.collinear.ai](https://platform.collinear.ai) (Developers → API Keys) to create or retrieve your API key.

## Error Responses

| Status | Meaning |
|--------|---------|
| `400` | Invalid request - see response for details |
| `401` | Invalid or missing API key |
| `403` | Key valid but insufficient permissions |
| `503` | Service temporarily unavailable |
