---
title: 'Evaluate Generated Data API'
openapi: 'POST /api/v1/synth_data/evaluate/{model_id}'
---

## Parameters

- **conversation_prefix**: A list of dictionaries representing the prompt (user input) in OpenAI format. Each dictionary should contain:
  - `role`: Either `"user"` or `"system"`
  - `content`: The corresponding text content

- **response**: A dictionary representing the assistant's response. Must follow OpenAI format:
  - `role`: `"assistant"`
  - `content`: The generated text to be evaluated

- **space_id**: The unique identifier for your Collinear workspace. [Learn more about obtaining a space ID](/get-started/get_space_id)

- **judge_id**: The unique identifier of the evaluation model to use. [Learn more about obtaining a judge ID](/get-started/get_judge_id)

## Headers

- **Authorization**: Bearer token for authenticating the API call
- **Content-Type**: `application/json`

## Request Example

```bash
curl --location 'https://api.collinear.ai/api/v1/synth_data/evaluate/{model_id}' \
--header 'Authorization: Bearer <your_token>' \
--header 'Content-Type: application/json' \
--data '{
  "conversation_prefix": [{
    "role": "user",
    "content": "How do actinomycetes differ from other types of bacteria in terms of their structure?"
  }],
  "response": {
    "role": "assistant",
    "content": "Actinomycetes differ from other bacteria in that they form filamentous chains of single cells..."
  },
  "space_id": "your_space_id",
  "judge_id": "your_judge_id"
}'
