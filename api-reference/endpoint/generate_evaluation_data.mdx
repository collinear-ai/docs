---
title: 'Generate Evaluation Data API'
openapi: 'POST /api/v1/synth_data/generate/{model_id}/{n}'
---

## Parameters

- **model_id**: UUID of the model to be used for generation. This defines the template/model logic applied to your input.

- **n**: Integer specifying how many synthetic data should be generated from the input file.

- **file**: The input data file (text, log, csv, json, chat transcript, etc.) to generate synthetic data from. This must be sent as a `multipart/form-data` upload.

## Headers

- **Authorization**: Bearer token for authenticating the API call.
- **Content-Type**: `multipart/form-data`

## Request Example

```bash
curl --location 'https://api.collinear.ai/api/v1/generate/32d7-485a-bea6-wyqgdqw67-gw/5' \
--header 'Authorization: Bearer <your_token>' \
--form 'file=@"path/to/your/input_file.txt"'
