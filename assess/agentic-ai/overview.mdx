Collinear's Agentic AI capabilities allow you to:

1. Accept various document types (text, logs, CSV, JSON, conversations) as input
2. Generate synthetic data based on the input
3. Assess the quality of generated data using multiple LLM judges

## Key Features

- **Input Flexibility**: Works with multiple file formats
- **Evaluation Data Generation**: Create realistic data based on your inputs
- **Comprehensive Evaluation**: Assess data across three dimensions:
  - Safety
  - Reliability
  - Performance

## Workflow

1. Upload your agents log in any format
2. Automatically generate evaluation data
3. Evaluate on the generated data using custom metrics
4. Review and export results
