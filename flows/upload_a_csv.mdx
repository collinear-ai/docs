title: Upload CSV And Run Judge
description: Use the Collinear AI Platform to run judge on your CSV.
---

# Step 1: Navigate to the judges dashboard
For uploading and testing a judge on a CSV file. We assume that you have already created a space and have a judge ready to be tested. If not, please refer to the [Create New Judge](/flows/create_new_judge) flow.
Click on the Test button to open the testing console.
<img
    style={{borderRadius: '0.5rem'}}
    src='/images/click_test_judge_button.png'
/>

# Step 2: Judge Testing Console
The testing console allows you to upload a CSV file and run the judge on it. You can also view the results of the judge evaluation.
<img
    style={{borderRadius: '0.5rem'}}
    src='/images/click_upload_csv.png'
/>

# Step 3: Upload CSV
Click on "Choose File" to select the CSV file you want to upload.
After selecting the file, click on "Judge CSV" to run the judge on the uploaded CSV file. [Download Sample CSV](https://docs.google.com/spreadsheets/d/e/2PACX-1vSd9GTIoC23e6w4JEECqkiAgA-sOwrmXYBa-bMeQ-23ofNayNKEnd7Ivp6-t_SNsZsWYsypWNN5PE_0/pub?output=csv)
<img
    style={{borderRadius: '0.5rem'}}
    src='/images/choose_file.png'
/>

# Step 4: View Results
After uploading the CSV file, you can view the results of the judge evaluation. The results will be displayed in a csv format.
Here is a sample output of the judge evaluation.[Download Output CSV](https://docs.google.com/spreadsheets/d/e/2PACX-1vRKwrTR0--6TjkW8PZ51AkaPR8wQu6IpD6492QmWpKBxZ_vOhz-aKjubfg-stXoM1LOr7W6ekWLy1hT/pub?output=csv)
