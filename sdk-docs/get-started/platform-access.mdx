## Why this matters

The [Collinear Platform](https://platform.collinear.ai) offers hosted assessment and visualizations for synthetic datasets. Before you can invoke the hosted Platform API or connect your Trait Mix runs to managed judges, you need a few identifiers from the Collinear dashboard:

- A **platform login** so you can reach your organization’s spaces
- At least one **Space** to scope datasets, judges, and runs
- A **Collinear Platform API key** (separate from the Trait Mix key used by the SDK)
- The **Space ID** that keeps uploads scoped to the right workspace
- One or more **Judge IDs** for scoring conversations through the API

This page shows how to obtain each identifier.

## Step 3 · Copy your Platform API key

**Why it’s important:** Hosted API endpoints (for uploads, judges, and synthetic runs) expect the `Authorization: Bearer ${COLLINEAR_API_KEY}` header. This key lives alongside your space, and it is distinct from the `TRAIT_BASIS_API_KEY` used by the SDK.

- Follow the updated [Platform API example](overview#platform-api-example) to upload the JSONL you generated.
- Jump into the [API reference](/api-reference/endpoint/judge_conversation) when you’re ready to script hosted evaluations.
